# Desafio 01: Criando sua primeira Landing Page com HTML e CSS

Primeiro desafio da Trilha de CSS da DIO! Nela, foi possível construir minha primeira Landing Page com HTML e CSS, colocando em prática os fundamentos do CSS,
as propriedades básicas da linguagem de estilização, além de trabalhar com as unidades de medidas relativas e absolutas que aprendi ao longo da trilha.

![image](https://user-images.githubusercontent.com/55519539/183538055-6cce606c-7d1d-4d15-a4be-ffeb5b37c956.png)

Como o foco deste projeto é praticar fundamentos do HTML e CSS, não foi aplicado conceitos de responsividade.
